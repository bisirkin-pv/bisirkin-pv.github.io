1. Установка

  • Для работы требуется Ruby 2+ и Python

  • Устанавливаем asciidoctor AsciiDoctor

gem install asciidoctor

  • Устанавливаем обработчик диаграмм asciidoctor-diagramm

gem install asciidoctor-diagram

  • Устанавливаем плагин для plantuml

gem install asciidoctor-plantuml

Для корректной работы требуется Graphviz

  • Устанавливаем модуль для Python link

pip install blockdiag

  • Запуск простого web сервера

Запускать из корневой папки хранения html.

Using Python
$ python -m SimpleHTTPServer 4242
Using Ruby >= 1.9.3
$ ruby -run -e httpd . -p 4242

2. Генерация HTML

Для корректной работы всех возможностей на каждой странице необходимо разместить следующий код:

:toc: (1)
:toc-title: Ссылки (2)
:toclevels: 4 (3)
:sectnums: (4)
:sectnumlevels: 3 (5)
:source-highlighter: coderay (6)
:imagesdir: images (7)
:iconsdir: icons (8)
:stylesdir: styles (9)
:icons: font (10)
:docinfodir: common (11)
:linkcss: (12)
:stem: latexmath (13)
:data-uri: (14)

Для того чтобы не приходилось всё это запоминать, данные теги вынесены в отдельный файл. Всё что требуется - это произвести его импорт(setting.adoc).

asciidoctor -r asciidoctor-diagram -a allow-uri-read -a nofooter index.adoc

3. Генерация PDF

Установка:

gem install asciidoctor-pdf --pre gem install concurrent-ruby

Для настройки используется файл конфигурации base-setting-pdf.yml

Для корректной работы с кирилицей необходимо подключить шрифт. Для этого скачивается сам файл tff и прописывается использование в настройках.

Выполнение:

asciidoctor -a nofooter -r asciidoctor-pdf -b pdf -r asciidoctor-diagram index.adoc

4. Отслеживание изменений и генерация html

Для отслеживания изменений нужно установить Guard

gem install guard guard-shell

Далее настраивается Guardfile расположенный в деректории наблюдения.

guard init

Guardfile
require 'asciidoctor'
require 'asciidoctor-diagram'

guard 'shell' do
  watch(/^index\.adoc$/) {|m|
    Asciidoctor.convert_file m[0], :require => 'asciidoctor-diagram'
  }
end

5. Генерация диаграмм

Для использования диаграмм необходимо размещать их код внутри блока и указывать название изображения(имя должно состоять из "названия страницы-тип диаграммы-рисунок") и тип изображения (svg). Так же для использования необходимо прописать папку для изображений.

[diagram-type, generated-file-name, generated-image-format]
....

6. Вывода блоков кода

6.1. Установка coderay

Установка:

gem install coderay

Для применения необходимоп прописать:

= Document Title
:source-highlighter: coderay

Для использования необходимо указать:

[source, type-syntax]
